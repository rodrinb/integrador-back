Productos

[GET] /productos/: Trae todos los productos

[POST] /productos/crearProducto: Crea el producto que pide la siguiente estructura

  • id: Number
  • title: String
  • image(Base64): String
  • price: Number
  • description: String
  • category: String (A, B o C)

[GET] /productos/:id : Trae el producto segun el _id (generado por MongoDB)

Usuarios

[GET] /usuarios/: Trae a todos los usuarios

[POST] /usuarios/registrar: Crea el usuario que pide la siguiente estructura

  • fullname: String
  • email: String
  • password: String

[POST] /usuarios/login: Inicia sesion del usuario, pide la siguiente estructura

  • email: String
  • password: String

[GET] /usuarios/:id : Trae el usuario segun el _id (generado por MongoDB)
